# E-commerce Platform for Local Shops

This repository contains the source code for the E-commerce Platform project,
developed as part of the Software Engineering course.

## Project Description
The platform allows local shops to publish products, users to browse items,
add them to cart, and place orders.

## Technologies
- Java
- Spring Boot
- MySQL
- REST API
- GitHub for version control

## Versioning
This project is versioned using GitHub.
All changes are tracked using git commit and git push.
